Hi, this is a cars reference app. Use the following commands:

  • GET /allCars - Getting JSON with all cars from the database
  • GET /allCarsByBrand?brand=*
  • GET /allCarsByColor?color=*
  • GET /someCar?number=* - Getting a JSON car by its number
  • DELETE /cars/delete?number=* - Deleting a car by its number
  • POST /cars/add - Adding a car to the database
  • GET /colors - get all colors from database
  • GET /brands - get all brands from database
